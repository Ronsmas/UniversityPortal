# Kisii University Course Registration

This project implements a simple course registration system for Kisii University, featuring a Spring Boot backend and a React frontend.

## How to Run the Backend

Navigate to the `backend` directory and run the Spring Boot application:

```bash
cd backend
mvn spring-boot:run
```

The backend API will be available at `http://localhost:8080`.

## How to Run the Frontend

Navigate to the `frontend` directory and start the React development server:

```bash
cd frontend
npm start
```

The frontend application will typically open in your browser at `http://localhost:3000`.

## Features

-   **Student Course Registration:** Students can register for available courses.
-   **Drop Course:** Students can drop registered courses.
-   **Kenyan Currency Support:** Course fees are displayed in Kenyan Shillings (KES).
-   **Duplicate Registration Prevention:** The system prevents students from registering for the same course multiple times.
